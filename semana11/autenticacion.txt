El login es el proceso mediante el cual un usuario ingresa a una aplicacion usando un nombre de usuario y una contraseña. Esto permite verificar su identidad antes de darle acceso al sistema. Algunas rutas deben estar protegidas porque contienen informacion privada o permiten realizar acciones importantes. Asi se evita que personas no autorizadas accedan a esos datos. No se deben exponer publicamente contraseñas, documentos de identidad, telefonos, direcciones, tokens, claves de API ni otra informacion sencible de los usuarios.
